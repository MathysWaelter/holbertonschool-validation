# First project

## DevOps using goHugo

## Prerequisites

* Different languages:

    > GNU Make in version 3.81+
    > Git (command line) in version 2+
    > Go Hugo v0.80+

## Lifecycle

* Different Make:

    > help: Show help command
    > build:  Create Website
    > clean:  Delete content
    > stop: Stop the process
    > run: Execute process
    > test: validate file
